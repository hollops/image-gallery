Image Gallery
A simple React + Vite image gallery project built with a category filter and image cards.

Features
Browse images in a responsive gallery layout
Filter images by category
Clean and modern UI
Tech Stack
React
Vite
CSS
Getting Started
Install dependencies
npm install
Start the development server
npm run dev
Open the local URL shown in the terminal
Environment Variables
Create a .env file in the project root and add any required API keys or configuration values.

Example:

VITE_API_KEY=your_api_key_here
Project Structure
src/ - React components and styles
public/ - Static assets
services/ - API-related logic
